### 在 Quant Wiki 中的应用

MIT 18.S096 / 18.642 与 Wiki 各章节高度对应，可以作为配套视频讲解：

- **第 2 讲（线性代数）** → 前置数学 [02 线性代数](../prerequisite-math/02-linear-algebra/)
- **第 3 讲（概率论）** → 前置数学 [03 概率统计](../prerequisite-math/03-probability/)
- **第 5–6 讲（随机过程 + 回归）** → 前置数学 [07 随机过程](../prerequisite-math/07-stochastic-processes/) 与 [04 数值方法](../prerequisite-math/04-numerical-methods/)
- **第 8–12 讲（时间序列）** → 前置数学 [06 时间序列](../prerequisite-math/06-time-series/)
- **第 14 讲（投资组合理论）** → 量化金融投资组合章节
- **第 17–21 讲（随机微积分 + BS公式）** → 量化金融期权定价章节
- **2024 版第 23 讲（ML）** → 前沿扩展，Wiki 尚未覆盖
