# 📚 量化学习体系

> 一条主线，从零到独立策略开发。不用想学什么——跟着走就行。

## 总览

```
Part 0: Python + 数学预备      ═══ 地基（约2周）
Part 1: 量化回测基础            ═══ 第一把剑（约1周）
Part 2: 因子与策略             ═══ 策略库（约3周）
Part 3: 回测与风险             ═══ 质检（约1周）
Part 4: ML 量化                ═══ 进阶武器（约2周）
Part 5: 实战冲刺               ═══ 毕业（约2周）
```

**每周 10 小时，预计 11 周完成主体系。** 每章 40-60 分钟，含自学 + 练习 + 掌握检查。

## 使用方法

1. **打开任何一章** → 按顺序学
2. **每章末尾有练习** — 选择题/填空题/编程题
3. **学完点「掌握检查」** → AI Teacher 出题验证
4. **通过 → 解锁下一章**

> 💡 这个页面不需要导航——直接用 AiTeacher 右下角气泡，它会告诉你当前学哪章。
