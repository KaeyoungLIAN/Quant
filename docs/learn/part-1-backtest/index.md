# Part 1: 量化回测基础

> 从零到一搭建完整的回测流程：数据获取、K 线构造、策略回放、绩效分析。学会用 Python 对任何想法做快速验证。

## 使用方法

1. **学本章内容** → 从数据到回测再到绩效指标的完整工作流
2. **跑代码** → 每一步都有可运行的 Jupyter Notebook
3. **做练习** → 修改参数、添加新逻辑，验证理解
4. **点「掌握检查」** → 解锁下一章

## 章节一览

| 章节 | 主题 | 内容简介 |
|------|------|----------|
| 1.1 从数据到K线 | 数据获取与预处理 | 用 yfinance 拉取行情数据，构造 OHLC K 线，处理缺失值、复权、时间对齐 |
| 1.2 第一个回测 | 回测引擎搭建 | 写一个最小回测框架：策略信号 → 持仓管理 → 成交记录，跑通完整闭环 |
| 1.3 绩效指标 | 回测结果评估 | 年化收益、最大回撤、夏普比率、胜率、盈亏比——每个指标的含义与 Python 实现 |

## 数据来源

所有章节使用 `yfinance` / `akshare` 获取 A 股或美股真实行情数据。

## 进度

每章预计 1-2 小时。全部 3 章约 3-6 小时完成。
