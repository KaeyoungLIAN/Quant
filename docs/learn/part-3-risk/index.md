# Part 3: 回测与风险

> 回测中最大的陷阱是"假信号"。本 Part 学习过拟合检测、交叉验证、风险分析框架与策略诊断——让回测结果可信，而不是自欺欺人。

## 使用方法

1. **学本章内容** → 从过拟合直觉到严谨的统计验证
2. **跑代码** → 对你的 Part 2 策略做"审问式"诊断
3. **做练习** → 用模拟数据或真实数据验证风险指标
4. **点「掌握检查」** → 解锁下一章

## 章节一览

| 章节 | 主题 | 内容简介 |
|------|------|----------|
| 3.1 过拟合与交叉验证 | 回测可信度检验 | Deflated Sharpe Ratio、组合交叉验证、回测过拟合概率（PBO）——量化你的策略是否只是"凑参数" |
| 3.2 风险分析框架 | 多维风险度量 | VaR/CVaR、最大回撤分布、压力测试、因子暴露分析——理解策略在不同市场环境下的脆弱性 |
| 3.3 策略诊断 | 归因与改进指南 | 收益归因（Brinson）、交易成本敏感性、存活者偏差检查、策略退化预警——系统化优化方法 |

## 数据来源

复用 Part 1-2 的行情与回测数据，增加模拟数据用于过拟合实验。

## 进度

每章预计 2-3 小时。全部 3 章约 1 周。
