# 第一章：出发

> 30分钟从零到画出第一张K线图。不用懂线性代数，不用懂金融——只要会打开浏览器。

## 一句话

**量化分析的起点不是数学，是跑通第一行代码，看到第一张图。**

## 本章结构

```
1.1 安装配置    → 装 Python + Jupyter，跑通第一行代码
1.2 第一次获取数据 → 用 yfinance 下载真实股票数据
1.3 画第一张K线图  → 这就是你要发朋友圈的那个瞬间
1.4 设定学习目标   → 停一停，问问自己为什么学量化
```

## 你需要什么

- 一台电脑（Windows / macOS / Linux 都行）
- 能上网
- **30分钟**的专注时间

没有编程经验？没关系。本章就是为**完全零基础**写的。

## 学完本章你能

- 在自己的电脑上搭建 Python 量化开发环境
- 从互联网获取任意股票的历史数据
- 画出专业的 K 线图（含成交量）
- 清晰定义自己的量化学习目标

> 🎯 目标不是学会所有东西——是**第一次体验到「我能做到」的感觉**。这比任何知识都重要。
