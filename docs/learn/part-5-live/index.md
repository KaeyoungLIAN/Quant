# Part 5: 实战冲刺

> 从回测到实盘——数据源对接、API 下单、资金管理，以及最终的毕业设计。把前四个 Part 的知识整合成一个可运行的量化系统。

## 使用方法

1. **学本章内容** → 回测与实盘的鸿沟在哪里，如何弥合
2. **跑代码** → 对接真实/模拟数据源，测试下单接口
3. **做练习** → 搭建迷你交易系统雏形
4. **完成毕设** → 选题、实现、回测、报告完整提交

## 章节一览

| 章节 | 主题 | 内容简介 |
|------|------|----------|
| 5.1 数据源与API | 实盘数据管道 | 聚宽/Tushare/Akshare 数据对接、交易所行情协议简介、K 线对齐与实时更新 |
| 5.2 执行与资金管理 | 下单与风控 | 市价/限价单模拟、滑点模型、凯利公式与固定比例仓位、最大回撤硬止损 |
| 5.3 毕业设计 | 完整的量化项目 | 选题指南、策略开发 → 回测 → 诊断 → 优化 → 报告的结构化流程，提供 5 个推荐选题 |

## 毕业设计选题示例

- A 股多因子选股 + 月度调仓系统
- 加密货币趋势跟踪 + 动态仓位管理
- 基于 XGBoost 因子的行业轮动策略
- 统计套利配对交易（期货/ETF）
- 你自己的策略想法（经导师确认）

## 数据来源

实盘/模拟数据源需自行注册账号（免费），本教程提供配置指南。

## 进度

每章预计 2-3 小时。毕设单独安排 1-2 周。全部约 2-3 周。
