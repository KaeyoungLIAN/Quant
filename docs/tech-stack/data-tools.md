## 各库详解

### AKShare

- **定位：** 目前国内最全面的开源金融数据接口库，覆盖 A 股、期货、期权、基金、债券、外汇、宏观经济、另类数据等
- **核心优势：** 社区活跃，更新频繁（每月多个版本），接口丰富（300+），全部免费
- **主要限制：** 部分接口速度较慢，数据格式偶尔变动
- **典型用途：** A 股行情、龙虎榜、资金流向、宏观经济数据采集

### Tushare

- **定位：** 国内老牌金融数据平台，覆盖全品类行情与基本面数据
- **核心优势：** 数据质量高，财务数据全面（三大报表、财务指标），支持港股/美股
- **主要限制：** 高级数据需要积分（付费），免费额度有限
- **典型用途：** A 股因子研究、财务分析、量化回测数据源

### yfinance

- **定位：** Yahoo Finance 的非官方 Python API，获取全球市场数据的最简方式
- **核心优势：** 零配置，一行代码获取全球股票历史数据
- **主要限制：** 依赖 Yahoo 非官方接口，稳定性不可靠（经常断连），分钟级数据有限
- **典型用途：** 美股快速原型验证、个人投资组合跟踪

### Alpha Vantage

- **定位：** 提供全球金融数据 API 的平台，Python SDK 封装
- **核心优势：** 数据覆盖全球，外汇和加密货币数据质量好
- **主要限制：** 免费版 5 次/分钟 API 限制，付费版较贵
- **典型用途：** 外汇分析、全球市场多品种数据获取

### rqdatac

- **定位：** 米筐科技 RiceQuant 的离线数据 SDK，专为量化策略开发设计
- **核心优势：** 与米筐回测平台无缝衔接，数据质量高，接口设计合理
- **主要限制：** 需要米筐账号，实盘数据付费
- **典型用途：** 米筐生态内的策略研发、回测与实盘

### financedatabase

- **定位：** 基于 Yahoo Finance 的全球上市公司基本面数据库
- **核心优势：** 提供结构化公司信息（行业、市值、报表等），方便筛选
- **主要限制：** 同样依赖 Yahoo，数据更新频率较低
- **典型用途：** 全球上市公司筛选、基本面初步分析

## 选型建议

| 需求 | 推荐 |
|------|------|
| **仅做 A 股** | AKShare（免费全面）或 Tushare（质量更高） |
| **A 股 + 美股 + 港股** | Tushare（一条龙）或 AKShare + yfinance 组合 |
| **仅做美股** | yfinance（最简单）或 Alpha Vantage（更稳定） |
| **米筐平台用户** | rqdatac（原生支持） |
| **全球上市公司筛选** | financedatabase |
| **财务数据深度研究** | Tushare（A 股） |
| **宏观经济数据** | AKShare（最丰富免费） |
