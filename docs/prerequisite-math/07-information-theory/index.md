# 07 信息论

> 信息论由 Claude Shannon 在 1948 年奠基，最初用于通信工程，如今是量化金融中不可或缺的工具——从最大熵资产定价到机器学习中的交叉熵损失，无处不在。

## 快速参考

| 概念 | 公式 | 量化金融应用 |
|------|------|-------------|
| 熵 $H(X)$ | $-\sum P(x)\log P(x)$ | 因子信息含量度量 |
| 交叉熵 $H(P,Q)$ | $-\sum P(x)\log Q(x)$ | 分类模型（信号预测）损失函数 |
| KL 散度 $D_{\text{KL}}(P\|Q)$ | $\sum P(x)\log\frac{P}{Q}$ | 模型近似误差、分布偏移监测 |
| 互信息 $I(X;Y)$ | $\sum P(x,y)\log\frac{P}{P(x)P(y)}$ | 非线性特征选择、因果发现 |
