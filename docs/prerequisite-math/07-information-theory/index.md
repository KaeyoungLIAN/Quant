# 07 信息论

> 信息论由 Claude Shannon 在 1948 年奠基，最初用于通信工程，如今是量化金融中不可或缺的工具——从最大熵资产定价到机器学习中的交叉熵损失，无处不在。

---

## 一、自信息与熵

### 1.1 自信息（Self-information）

对于一个离散随机变量 $X$，取值 $x$ 的概率为 $P(x)$，其**自信息**定义为：

$$
I(x) = -\log_2 P(x) \quad (\text{单位：比特})
$$

- 概率越小的**事件**，发生时的信息量越大
- 确定事件（$P=1$）的信息量为 $0$

### 1.2 熵（Entropy）

**熵**是随机变量 $X$ 的平均自信息量：

$$
H(X) = -\sum_{x \in \mathcal{X}} P(x) \log_2 P(x)
$$

熵衡量了一个分布的不确定性——熵越大，不确定性越高。

### 1.3 手算实例：偏倚硬币的熵

一枚硬币 $P(\text{正面}) = 0.7$，$P(\text{反面}) = 0.3$。

| 事件 | 概率 $P$ | $\log_2 P$ | $-P \log_2 P$ |
|------|----------|------------|---------------|
| 正面 | $0.7$ | $\log_2 0.7 = -0.5146$ | $0.7 \times 0.5146 = 0.3602$ |
| 反面 | $0.3$ | $\log_2 0.3 = -1.7370$ | $0.3 \times 1.7370 = 0.5211$ |
| **合计** | $1.0$ | — | **$H = 0.8813$ 比特** |

对比：一枚**公平**硬币（$P=0.5$）的熵为 $H = -[0.5\log_2 0.5 + 0.5\log_2 0.5] = 1$ 比特。偏倚降低了不确定性。

> **Quant Link**：在**因子模型**中，我们可以用熵来衡量一个因子的信息含量——因子收益率的分布越分散（熵越大），该因子携带的信息越多。反之，如果某个因子几乎总是正收益，其熵接近 0，说明它可能已被市场充分定价。

## 二、交叉熵与 KL 散度

### 2.1 交叉熵（Cross-Entropy）

交叉熵衡量用分布 $Q$ 来编码来自分布 $P$ 的事件所需的平均比特数：

$$
H(P, Q) = -\sum_{x} P(x) \log Q(x)
$$

### 2.2 KL 散度（Kullback-Leibler Divergence）

KL 散度衡量用 $Q$ 近似 $P$ 带来的额外信息损失：

$$
D_{\text{KL}}(P \parallel Q) = \sum_{x} P(x) \log \frac{P(x)}{Q(x)} = H(P, Q) - H(P)
$$

性质：
- $D_{\text{KL}}(P \parallel Q) \ge 0$，当且仅当 $P = Q$ 时取等
- 不是对称的：$D_{\text{KL}}(P \parallel Q) \neq D_{\text{KL}}(Q \parallel P)$

### 2.3 手算实例：KL 散度（3 种结果）

真实分布 $P$ 与模型分布 $Q$：

| 结果 | $P(x)$ | $Q(x)$ | $\log(P/Q)$ | $P \cdot \log(P/Q)$ |
|------|--------|--------|-------------|---------------------|
| A | $0.50$ | $0.60$ | $\log(0.5/0.6) = -0.1823$ | $0.50 \times (-0.1823) = -0.0912$ |
| B | $0.30$ | $0.25$ | $\log(0.3/0.25) = 0.1823$ | $0.30 \times 0.1823 = 0.0547$ |
| C | $0.20$ | $0.15$ | $\log(0.2/0.15) = 0.2877$ | $0.20 \times 0.2877 = 0.0575$ |
| **合计** | $1.0$ | $1.0$ | — | **$D_{\text{KL}} = 0.0210$ 比特** |

$D_{\text{KL}}(P \parallel Q) = 0.0210$ 比特，非常小——说明模型 $Q$ 对真实分布 $P$ 的近似做得不错。

### 2.4 交叉熵损失函数

在机器学习分类任务中，对于真实标签 $y$ 和预测概率 $\hat{y}$，**交叉熵损失**为：

$$
\mathcal{L} = -\sum_{i} y_i \log \hat{y}_i
$$

> **Quant Link**：在**信用风险建模**中，我们用交叉熵损失训练违约概率（PD）模型。如果一个贷款申请人的真实标签为违约（$y=1$），模型预测 $\hat{y}=0.05$，则损失为 $-\log(0.05) \approx 2.996$——模型越"自信"地犯错，惩罚越大。

> **Quant Link**：**最大熵原理**（Maximum Entropy Principle）认为，在所有满足约束条件的分布中，应选择熵最大的那一个（即最无偏、最保守的分布）。在**资产定价**中，Avellaneda 等人用最大熵方法从期权价格中**隐含地提取风险中性分布**——仅以观察到的期权市场价格为约束，最大化熵得到最无偏的定价测度。

## 三、互信息（Mutual Information）

### 3.1 定义

互信息衡量两个随机变量 $X$ 和 $Y$ 之间的**共享信息量**：

$$
I(X; Y) = \sum_{x \in \mathcal{X}} \sum_{y \in \mathcal{Y}} P(x, y) \log \frac{P(x, y)}{P(x)P(y)}
$$

等价于：

$$
I(X; Y) = H(X) - H(X \mid Y) = H(Y) - H(Y \mid X)
$$

### 3.2 手算实例：互信息

考虑一个二元随机变量的联合分布 $(X, Y)$：

| $P(x, y)$ | $Y=0$ | $Y=1$ | $P(X)$ |
|-----------|-------|-------|--------|
| $X=0$ | $0.3$ | $0.1$ | $0.4$ |
| $X=1$ | $0.2$ | $0.4$ | $0.6$ |
| $P(Y)$ | $0.5$ | $0.5$ | $1.0$ |

**步骤 1**：计算 $P(x)P(y)$

| $x$ | $y$ | $P(x,y)$ | $P(x)P(y)$ | $\log\frac{P(x,y)}{P(x)P(y)}$ | 贡献 |
|-----|-----|----------|------------|-------------------------------|------|
| 0 | 0 | $0.3$ | $0.4 \times 0.5 = 0.2$ | $\log(0.3/0.2) = 0.5850$ | $0.3 \times 0.5850 = 0.1755$ |
| 0 | 1 | $0.1$ | $0.4 \times 0.5 = 0.2$ | $\log(0.1/0.2) = -1.0000$ | $0.1 \times (-1.0000) = -0.1000$ |
| 1 | 0 | $0.2$ | $0.6 \times 0.5 = 0.3$ | $\log(0.2/0.3) = -0.5850$ | $0.2 \times (-0.5850) = -0.1170$ |
| 1 | 1 | $0.4$ | $0.6 \times 0.5 = 0.3$ | $\log(0.4/0.3) = 0.4150$ | $0.4 \times 0.4150 = 0.1660$ |

**步骤 2**：求和

$$
I(X; Y) = 0.1755 - 0.1000 - 0.1170 + 0.1660 = 0.1245 \text{ 比特}
$$

$X$ 和 $Y$ 之间存在**正**互信息——知道其中一个变量能减少约 0.1245 比特的不确定性。

> **Quant Link**：**特征选择**中，互信息是比相关系数更强大的指标——它能捕捉**非线性**依赖。在量化因子研究中，我们计算每个候选因子与未来收益的互信息，挑选出信息含量最高的因子，而非仅仅依赖线性相关系数。

## 四、信息论的量化金融应用

### 4.1 最大熵资产定价

给定 $N$ 只看涨期权的市场价格 $C_i$，最大熵方法求解以下优化问题：

$$
\max_{q} H(q) = -\sum_{j} q_j \log q_j
\quad \text{s.t.} \quad \sum_{j} q_j (S_j - K_i)^+ = C_i,\; \sum_j q_j = 1
$$

其中 $q_j$ 是到期时资产价格 $S_j$ 的风险中性概率，$(S_j - K_i)^+$ 为行权价 $K_i$ 的期权收益。这给出了**最无偏**的隐含分布。

### 4.2 交叉熵在回测中的作用

在机器学习驱动的量化策略中，交叉熵损失是分类型交易信号（涨/跌/平）的标准损失函数：

$$
\mathcal{L}_{\text{CE}} = -\frac{1}{T} \sum_{t=1}^T \sum_{c=1}^C y_{t,c} \log \hat{y}_{t,c}
$$

其中 $C$ 是信号类别数（如上涨、下跌、横盘），$y_{t,c}$ 为 one-hot 真实标签。

## 五、Python 示例：熵和 KL 散度

```python
import numpy as np

def entropy(p):
    """计算离散分布的熵（单位：比特）"""
    p = np.asarray(p)
    return -np.sum(p * np.log2(p + 1e-15))  # 加极小值防止 log(0)

def kl_divergence(p, q):
    """计算 KL(P || Q)，单位：比特"""
    p = np.asarray(p)
    q = np.asarray(q)
    return np.sum(p * np.log2((p + 1e-15) / (q + 1e-15)))

# --- 偏倚硬币熵 ---
p_coin = [0.7, 0.3]
print(f"偏倚硬币熵: H = {entropy(p_coin):.4f} 比特")
# 输出: 偏倚硬币熵: H = 0.8813 比特

# 对比公平硬币
p_fair = [0.5, 0.5]
print(f"公平硬币熵: H = {entropy(p_fair):.4f} 比特")
# 输出: 公平硬币熵: H = 1.0000 比特

# --- KL 散度（3 结果）---
p = np.array([0.50, 0.30, 0.20])
q = np.array([0.60, 0.25, 0.15])

kl = kl_divergence(p, q)
print(f"KL(P||Q) = {kl:.4f} 比特")
# 输出: KL(P||Q) = 0.0210 比特

# --- 交叉熵 ---
cross_entropy = entropy(p) + kl
print(f"交叉熵 H(P,Q) = {cross_entropy:.4f} 比特")
# 输出: 交叉熵 H(P,Q) ≈ 1.5717 比特

# --- 互信息 ---
def mutual_information(joint):
    """计算互信息 I(X;Y)"""
    joint = np.asarray(joint)
    px = joint.sum(axis=1)  # P(X)
    py = joint.sum(axis=0)  # P(Y)
    # 外积 P(x)P(y)
    outer = np.outer(px, py)
    # 逐元素计算
    mi = np.sum(joint * np.log2((joint + 1e-15) / (outer + 1e-15)))
    return mi

joint = np.array([[0.3, 0.1], [0.2, 0.4]])
mi = mutual_information(joint)
print(f"互信息 I(X;Y) = {mi:.4f} 比特")
# 输出: 互信息 I(X;Y) = 0.1245 比特
```

## 小结

| 概念 | 公式 | 量化金融应用 |
|------|------|-------------|
| 熵 $H(X)$ | $-\sum P(x)\log P(x)$ | 因子信息含量度量 |
| 交叉熵 $H(P,Q)$ | $-\sum P(x)\log Q(x)$ | 分类模型（信号预测）损失函数 |
| KL 散度 $D_{\text{KL}}(P\|Q)$ | $\sum P(x)\log\frac{P}{Q}$ | 模型近似误差、分布偏移监测 |
| 互信息 $I(X;Y)$ | $\sum P(x,y)\log\frac{P}{P(x)P(y)}$ | 非线性特征选择、因果发现 |
