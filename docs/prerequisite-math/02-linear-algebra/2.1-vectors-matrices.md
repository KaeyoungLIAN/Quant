# 2.1 向量与矩阵

> 线代从向量和矩阵开始。它们是量化金融中"组织数据"和"描述关系"的基本语言。

---

## 一、向量

### 1.1 定义

向量是既有大小又有方向的量，通常写成列向量：

$$
\boldsymbol{v} = \begin{bmatrix} v_1 \\ v_2 \\ \vdots \\ v_n \end{bmatrix}
$$

或行向量：$\boldsymbol{v}^T = [v_1, v_2, \dots, v_n]$

在量化中，向量最常见的用法是**表示一个资产组合的各资产权重**，或**一组因子暴露**。

::: tip Quant Link
一个投资组合的权重分布就是一个向量 $\boldsymbol{w} = (w_1, w_2, \dots, w_n)^T$，其中 $w_i$ 是第 $i$ 只资产的占比，且 $\sum w_i = 1$。
:::

### 1.2 基本运算

**加法**：$\boldsymbol{u} + \boldsymbol{v} = (u_1 + v_1, \dots, u_n + v_n)^T$

**数乘**：$c \cdot \boldsymbol{v} = (c v_1, \dots, c v_n)^T$

**点积（内积）**：

$$
\boldsymbol{u} \cdot \boldsymbol{v} = \sum_{i=1}^n u_i v_i
$$

**几何意义**：$\boldsymbol{u} \cdot \boldsymbol{v} = \|\boldsymbol{u}\| \|\boldsymbol{v}\| \cos \theta$，其中 $\theta$ 是两向量夹角。

点积为正 → 方向相近；点积为负 → 方向相反；点积为零 → 正交（无关）。

::: tip Quant Link
**相关性**就是标准化后的点积：$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$。相关性为 0 就是正交——两只资产的收益率互不相关，这是组合分散化的理论基础。
:::

### 1.3 范数

向量的长度称为**范数**。

**L2 范数（欧几里得距离）**：

$$
\|\boldsymbol{v}\|_2 = \sqrt{v_1^2 + v_2^2 + \dots + v_n^2}
$$

**L1 范数**（绝对值之和）：

$$
\|\boldsymbol{v}\|_1 = |v_1| + |v_2| + \dots + |v_n|
$$

::: tip Quant Link
L1 范数在**套利定价模型**中用于正则化（Lasso 回归），迫使部分因子系数变为零，实现因子选择。L2 范数对应 Ridge 回归。
:::

## 二、矩阵

### 2.1 定义

矩阵是矩形的数字表：

$$
A = \begin{bmatrix}
a_{11} & a_{12} & \dots & a_{1n} \\
a_{21} & a_{22} & \dots & a_{2n} \\
\vdots & \vdots & \ddots & \vdots \\
a_{m1} & a_{m2} & \dots & a_{mn}
\end{bmatrix}
$$

**维度**：$m \times n$（$m$ 行 $n$ 列）。若 $m = n$，称为**方阵**。

### 2.2 基本运算

**加法**：同维度矩阵逐元素相加。

**数乘**：常数乘以矩阵的每个元素。

**乘法** $A_{m \times n} \cdot B_{n \times p} = C_{m \times p}$：

$$
c_{ij} = \sum_{k=1}^n a_{ik} \cdot b_{kj}
$$

**重点**：矩阵乘法不满足交换律——$AB \neq BA$（一般情况）。

::: tip Quant Link
**因子模型**：资产收益率矩阵 $R_{T \times N}$（$T$ 天，$N$ 资产）= 因子暴露矩阵 $X_{T \times K}$ × 因子收益率 $F_{K \times N}$ + 残差。矩阵乘法就是"用因子解释收益"。
:::

### 2.3 特殊矩阵

| 类型 | 定义 | 量化用途 |
|------|------|----------|
| **单位矩阵** $I$ | 对角线上为 1，其余为 0 | 矩阵的"1" |
| **对角矩阵** | 只有主对角线上非零 | 资产独立方差假设 |
| **对称矩阵** | $A^T = A$ | 协方差矩阵 |
| **转置** $A^T$ | 行列互换 | 行向量 ↔ 列向量 |
| **迹 (trace)** | $\text{tr}(A) = \sum a_{ii}$ | 组合方差计算 |

### 2.4 矩阵的逆

若存在 $B$ 使得 $AB = BA = I$，则 $B = A^{-1}$ 是 $A$ 的逆矩阵。

**可逆的条件**：方阵且行列式不为零（满秩）。

$$(AB)^{-1} = B^{-1}A^{-1}$$

::: tip Quant Link
**组合优化**中，求最优权重需要计算协方差矩阵的逆 $\boldsymbol{w}^* = \frac{\Sigma^{-1} \boldsymbol{1}}{\boldsymbol{1}^T \Sigma^{-1} \boldsymbol{1}}$。协方差矩阵奇异（不可逆）时，需要使用伪逆或正则化。
:::

## 三、Python 实践

```python
import numpy as np

# 向量
v = np.array([0.3, 0.5, 0.2])  # 权重向量
print(f"权重之和: {v.sum():.2f}")  # 1.00

# 点积
u = np.array([1, 2, 3])
v = np.array([4, 5, 6])
dot = np.dot(u, v)
print(f"点积: {dot}")  # 32

# 范数
norm_l2 = np.linalg.norm(v)
norm_l1 = np.sum(np.abs(v))
print(f"L2范数: {norm_l2:.4f}, L1范数: {norm_l1}")

# 矩阵乘法
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
print(f"矩阵乘法:\n{A @ B}")

# 矩阵的逆
A_inv = np.linalg.inv(A)
print(f"逆矩阵:\n{A_inv}")
print(f"验证 A @ A⁻¹:\n{A @ A_inv}")
```

## 小结

| 概念 | 要点 |
|------|------|
| 向量 | 一组有序数，表示权重、因子暴露 |
| 点积 | 衡量两向量的相关性 |
| 范数 | 向量长度的度量，L1→Lasso，L2→Ridge |
| 矩阵 | 数字表，组织数据和描述变换 |
| 逆矩阵 | 方程求解和组合优化的核心工具 |
