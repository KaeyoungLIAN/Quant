# 2.6 矩阵微积分

> 当你对向量或矩阵求导时，就是矩阵微积分。这是机器学习反向传播和量化组合优化的数学基础。

---

## 一、标量对向量的导数

### 1.1 梯度

设 $f: \mathbb{R}^n \to \mathbb{R}$ 是标量函数，其对向量 $\boldsymbol{x}$ 的导数（梯度）为：

$$
\frac{\partial f}{\partial \boldsymbol{x}} = \nabla f = \begin{bmatrix}
\frac{\partial f}{\partial x_1} \\
\frac{\partial f}{\partial x_2} \\
\vdots \\
\frac{\partial f}{\partial x_n}
\end{bmatrix}
$$

### 1.2 常用公式

**线性函数**：$f(\boldsymbol{x}) = \boldsymbol{a}^T \boldsymbol{x}$

$$
\frac{\partial}{\partial \boldsymbol{x}} (\boldsymbol{a}^T \boldsymbol{x}) = \boldsymbol{a}
$$

**二次型**：$f(\boldsymbol{x}) = \boldsymbol{x}^T A \boldsymbol{x}$（$A$ 对称）

$$
\frac{\partial}{\partial \boldsymbol{x}} (\boldsymbol{x}^T A \boldsymbol{x}) = 2A\boldsymbol{x}
$$

**平方和（最小二乘）**：$f(\boldsymbol{x}) = \|A\boldsymbol{x} - \boldsymbol{b}\|_2^2$

$$
\frac{\partial f}{\partial \boldsymbol{x}} = 2A^T (A\boldsymbol{x} - \boldsymbol{b})
$$

> 令此梯度为零即得到正规方程 $A^T A \boldsymbol{x} = A^T \boldsymbol{b}$。

::: tip Quant Link
**组合优化**：最小化组合方差 $\sigma_p^2 = \boldsymbol{w}^T \Sigma \boldsymbol{w}$ 对权重 $\boldsymbol{w}$ 求导，得到最优条件 $\Sigma \boldsymbol{w} = \lambda \boldsymbol{1}$，就是拉格朗日乘子法的结果。
:::

## 二、标量对矩阵的导数

### 2.1 定义

设 $f: \mathbb{R}^{m \times n} \to \mathbb{R}$，对矩阵 $X$ 的导数为：

$$
\frac{\partial f}{\partial X}_{ij} = \frac{\partial f}{\partial x_{ij}}
$$

### 2.2 常用公式

$$
\frac{\partial}{\partial X} \text{tr}(AX) = A^T
$$

$$
\frac{\partial}{\partial X} \det(X) = \det(X) (X^{-1})^T
$$

$$
\frac{\partial}{\partial X} \boldsymbol{x}^T A \boldsymbol{x} = 2A\boldsymbol{x} \quad (\text{对向量})
$$

---

## 三、矩阵微积分的量化应用

### 3.1 线性回归的解析解

向量形式的目标函数：

$$
L(\boldsymbol{\beta}) = \| \boldsymbol{y} - X\boldsymbol{\beta} \|_2^2
$$

求梯度并令为零：

$$
\frac{\partial L}{\partial \boldsymbol{\beta}} = -2X^T(\boldsymbol{y} - X\boldsymbol{\beta}) = 0
$$

得 $\hat{\boldsymbol{\beta}} = (X^T X)^{-1} X^T \boldsymbol{y}$。

### 3.2 神经网络反向传播

对 $\boldsymbol{x}^T \boldsymbol{x}$ 求导不过是 $2\boldsymbol{x}$，但层层叠加就是链式法则 + 矩阵微积分——整篇文章的复杂计算。

### 3.3 Markowitz 优化

目标：$\min \boldsymbol{w}^T \Sigma \boldsymbol{w}$，约束 $\boldsymbol{1}^T \boldsymbol{w} = 1$

拉格朗日函数：$\mathcal{L} = \boldsymbol{w}^T \Sigma \boldsymbol{w} - \lambda (\boldsymbol{1}^T \boldsymbol{w} - 1)$

对 $\boldsymbol{w}$ 求导：$2\Sigma \boldsymbol{w} - \lambda \boldsymbol{1} = 0 \Rightarrow \boldsymbol{w} = \frac{\lambda}{2} \Sigma^{-1} \boldsymbol{1}$

带入约束解得 $\lambda$，得到最优权重。

## 四、Python 实践

```python
import sympy as sp
import numpy as np

# 符号验证：对向量求导
x1, x2 = sp.Symbol('x1'), sp.Symbol('x2')
x = sp.Matrix([x1, x2])
A = sp.Matrix([[2, 1], [1, 3]])
b = sp.Matrix([1, 2])

# 二次型求导
f = x.T * A * x
grad = [sp.diff(f[0], var) for var in (x1, x2)]
print(f"∂(xᵀAx)/∂x = {grad}")  # [4*x1 + 2*x2, 2*x1 + 6*x2]
print(f"验证 = 2Ax = {2 * A * x}")

# 最小二乘
beta = sp.Matrix([sp.Symbol('b1'), sp.Symbol('b2')])
X = sp.Matrix([[1, 1], [1, 2], [1, 3]])
y = sp.Matrix([2.1, 3.9, 6.0])
loss = (y - X * beta).T * (y - X * beta)
grad_beta = [sp.diff(loss[0], var) for var in beta]
print(f"\n最小二乘梯度 = {grad_beta}")
# 解正规方程
XtX = X.T * X
Xty = X.T * y
beta_opt = XtX.inv() * Xty
print(f"最优 β = {beta_opt}")
```

## 小结

| 场景 | 公式 | 量化应用 |
|------|------|----------|
| 线性函数 | $\frac{\partial}{\partial \boldsymbol{x}} \boldsymbol{a}^T \boldsymbol{x} = \boldsymbol{a}$ | 因子模型中的暴露度 |
| 二次型 | $\frac{\partial}{\partial \boldsymbol{x}} \boldsymbol{x}^T A \boldsymbol{x} = 2A\boldsymbol{x}$ | Markowitz 组合方差 |
| 最小二乘 | $2X^T (X \boldsymbol{\beta} - \boldsymbol{y})$ | 线性回归 |
| 矩阵求导 | $\frac{\partial}{\partial X} \text{tr}(AX) = A^T$ | 神经网络反向传播 |
