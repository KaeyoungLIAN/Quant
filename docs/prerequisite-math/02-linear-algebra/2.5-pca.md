# 2.5 PCA

> PCA（主成分分析）是量化中使用频率最高的降维工具之一。

---

## 一、PCA 的核心思想

**目标**：用少量的"综合变量"（主成分）来解释原始数据中大部分的方差。

PCA 所做的就是：找到数据方差最大的方向（第一主成分），然后找与之正交的次大方向（第二主成分），依此类推。

数学上，PCA 等价于对协方差矩阵做特征值分解：

$$
\Sigma = Q \Lambda Q^T
$$

- 特征向量 $\boldsymbol{q}_1$ 是**第一主成分方向**（方差 $\lambda_1$ 最大）
- 特征向量 $\boldsymbol{q}_2$ 是**第二主成分方向**（方差 $\lambda_2$ 次大）
- 以此类推

## 二、PCA 步骤

1. **标准化数据**（减去均值，除以标准差）
2. **计算协方差矩阵**
3. **特征值分解**
4. **选择前 $k$ 个主成分**
5. **将原始数据投影到主成分空间**

## 三、量化应用

### 3.1 利率曲线分解

市场利率曲线的变动，通常可以被前 3 个主成分解释 95% 以上：

| 主成分 | 名称 | 解释比例 | 含义 |
|--------|------|----------|------|
| PC1 | **水平** (Level) | ~70% | 所有利率同时升降 |
| PC2 | **倾斜** (Slope) | ~20% | 短端和长端反向变动 |
| PC3 | **曲率** (Curvature) | ~5% | 中间段相对两端变动 |

这对**固定收益投资组合的风险管理**意义重大——不需要跟踪所有期限的利率，只需要控制这 3 个主成分的暴露。

### 3.2 股票因子分析

从大量股票收益率中提取"隐藏因子"——比如规模因子、价值因子、动量因子等。这些因子就是股票协方差矩阵的主成分。

### 3.3 协方差矩阵去噪

样本协方差矩阵包含大量噪声。PCA 后只保留前 $k$ 个主成分（信号），用截断后的矩阵做组合优化，效果往往比原始矩阵好。

```python
import numpy as np

def pca_denoise(Sigma, k):
    """用 PCA 对协方差矩阵去噪"""
    eigvals, eigvecs = np.linalg.eigh(Sigma)
    # 只保留前 k 个最大的特征值
    eigvals_filtered = eigvals.copy()
    eigvals_filtered[:-k] = 0
    return eigvecs @ np.diag(eigvals_filtered) @ eigvecs.T
```

## 四、Python 实践

```python
import numpy as np
from sklearn.decomposition import PCA

# 模拟 100 只股票、500 天的收益率数据
np.random.seed(42)
n_stocks, n_days = 100, 500
returns = np.random.randn(n_days, n_stocks)

# 用 sklearn 做 PCA
pca = PCA(n_components=10)
pca.fit(returns)

# 方差解释比例
var_ratio = pca.explained_variance_ratio_
print("前10个主成分方差解释比例:")
for i, r in enumerate(var_ratio):
    print(f"  PC{i+1}: {r:.4f} ({var_ratio[:i+1].sum():.4f})")

# 手动实现 PCA
cov = np.cov(returns.T)
eigvals, eigvecs = np.linalg.eigh(cov)
# 排序（从大到小）
idx = np.argsort(eigvals)[::-1]
eigvals, eigvecs = eigvals[idx], eigvecs[:, idx]
print(f"\n第一主成分方向（前5个权重）: {eigvecs[:5, 0]}")
```

## 小结

| 概念 | 要点 |
|------|------|
| PCA 本质 | 协方差矩阵的特征值分解 |
| 主成分 | 方差最大的方向（特征向量） |
| 方差解释 | 特征值大小决定重要性 |
| 量化应用 | 利率曲线分解、因子提取、协方差去噪 |
