# 2.3 正定矩阵与二次型

> 正定矩阵在优化和风险管理中无处不在——它决定了"这个函数是凸的还是凹的"、"风险到底有多大"。

---

## 一、二次型

由对称矩阵 $A$ 定义的二次型：

$$
Q(\boldsymbol{x}) = \boldsymbol{x}^T A \boldsymbol{x} = \sum_{i=1}^n \sum_{j=1}^n a_{ij} x_i x_j
$$

**例**：$A = \begin{bmatrix} 2 & 1 \\ 1 & 3 \end{bmatrix}$，则

$$
Q(\boldsymbol{x}) = [x_1, x_2] \begin{bmatrix} 2 & 1 \\ 1 & 3 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} = 2x_1^2 + 2x_1 x_2 + 3x_2^2
$$

::: tip Quant Link
**组合方差**就是二次型：$\sigma_p^2 = \boldsymbol{w}^T \Sigma \boldsymbol{w}$，其中 $\Sigma$ 是协方差矩阵（对称正定）。
:::

---

## 二、正定矩阵

### 2.1 定义

对称矩阵 $A$ 称为：
- **正定**：对任意 $\boldsymbol{x} \neq \boldsymbol{0}$，$\boldsymbol{x}^T A \boldsymbol{x} > 0$
- **半正定**：对任意 $\boldsymbol{x}$，$\boldsymbol{x}^T A \boldsymbol{x} \ge 0$
- **负定**：对任意 $\boldsymbol{x} \neq \boldsymbol{0}$，$\boldsymbol{x}^T A \boldsymbol{x} < 0$

### 2.2 判定方法

1. **特征值法**：所有特征值 > 0 ⇔ 正定
2. **顺序主子式法**：所有顺序主子式 > 0 ⇔ 正定
3. **Cholesky 分解法**：$A = LL^T$ 存在 ⇔ 正定

### 2.3 为什么重要

在优化中，海森矩阵 $H$ 的正定性决定了极值类型：
- $H$ 正定 → 局部最小值
- $H$ 负定 → 局部最大值

在风险管理中，所有协方差矩阵都是**半正定**的——因为方差不能为负。

::: tip Quant Link
为什么协方差矩阵一定是半正定的？因为组合方差 $\boldsymbol{w}^T \Sigma \boldsymbol{w} = \text{Var}(\boldsymbol{w}^T R) \ge 0$（方差不可能为负）。这保证了最小方差组合的存在性和唯一性。

如果按 $\Sigma$ 不正定（即奇异），说明存在冗余资产——有些资产可以被其他资产线性组合复制，组合优化时需要处理。
:::

## 三、Cholesky 分解

正定矩阵可以分解为 $A = LL^T$，其中 $L$ 是下三角矩阵。

Cholesky 分解在量化中最直接的应用：**生成相关随机数**。

假设你有一个 $n$ 维正态分布 $N(\boldsymbol{0}, \Sigma)$，如何生成样本？

1. 对 $\Sigma$ 做 Cholesky 分解：$\Sigma = LL^T$
2. 生成 $n$ 个独立标准正态变量 $\boldsymbol{z}$
3. 计算 $\boldsymbol{x} = L \boldsymbol{z}$

则 $\boldsymbol{x} \sim N(\boldsymbol{0}, \Sigma)$。

```python
import numpy as np

# 协方差矩阵
Sigma = np.array([[1.0, 0.7], [0.7, 1.0]])
L = np.linalg.cholesky(Sigma)
print(f"Cholesky 分解 L:\n{L}")

# 生成相关随机数
np.random.seed(42)
z = np.random.normal(size=(1000, 2))  # 独立标准正态
x = z @ L.T  # 转化为有相关性
print(f"样本相关系数: {np.corrcoef(x.T)[0,1]:.4f}")  # 接近 0.7
```

---

## 小结

| 概念 | 要点 |
|------|------|
| 二次型 | $\boldsymbol{x}^T A \boldsymbol{x}$ |
| 正定矩阵 | 所有特征值 > 0 |
| 半正定 | 所有协方差矩阵都是半正定的 |
| Cholesky 分解 | $A = LL^T$，唯一存在条件就是 $A$ 正定 |
| 量化应用 | 组合方差、相关随机数生成、凸优化 |
