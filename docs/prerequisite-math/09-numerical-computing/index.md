# 09 数值计算

> 数值计算是连接数学理论和实际代码的桥梁。量化金融中绝大多数模型没有解析解——从期权定价到风险度量，全靠数值方法。理解浮点误差、收敛速度和数值稳定性，是写出可靠交易系统的前提。

---

## 一、浮点数与精度问题

### 1.1 机器 Epsilon

**机器 epsilon** ($\varepsilon_m$) 是计算机浮点数系统中，满足 $1.0 + \varepsilon \neq 1.0$ 的最小正数。在 IEEE 754 标准中：

| 精度 | 类型 | 机器 epsilon | 有效位数 |
|------|------|-------------|----------|
| 单精度 | float32 | $2^{-23} \approx 1.19 \times 10^{-7}$ | ~7 位 |
| 双精度 | float64 | $2^{-52} \approx 2.22 \times 10^{-16}$ | ~16 位 |

Python 中可用以下方式验证：

```python
eps = 1.0
while 1.0 + eps != 1.0:
    eps /= 2.0
print(f"机器 epsilon = {eps * 2}")  # 输出: 2.220446049250313e-16
```

### 1.2 灾难性抵消（Catastrophic Cancellation）

当两个大小相近的数相减时，有效数字大量丢失——这是数值计算中最危险的误差来源之一。

### 1.3 手算实例：$(10^{16} + 1) - 10^{16}$

双精度浮点数只有约 16 位有效数字。$10^{16}$ 在双精度下表示为 $1.0 \times 10^{16}$。

| 步骤 | 数学运算 | 计算机实际发生了什么 |
|------|----------|---------------------|
| 1 | $10^{16} + 1$ | $10^{16}$ 的第 17 位是 1，但双精度只有 ~16 位，**1 被截断** |
| 2 | 结果 | 计算机认为 $10^{16} + 1 = 10^{16}$ |
| 3 | $(10^{16} + 1) - 10^{16}$ | $10^{16} - 10^{16} = 0$ |
| 4 | 理论正确结果 | **$1$** |

验证：

```python
print((10**16 + 1) - 10**16)  # 输出: 0.0
```

**如何避免？** 用 `math.fsum` 或重新排列计算顺序。例如，如果计算 $a - b$ 且 $a \approx b$，尝试代数变换：

$$
\frac{1}{\sqrt{x + \delta} - \sqrt{x}} \quad\rightarrow\quad \frac{\sqrt{x + \delta} + \sqrt{x}}{\delta}
$$

> **Quant Link**：在**计算期权净值**时，如果两个非常大的头寸方向相反（如一个看涨 + 执行价 100 的空头 + 一个执行价 101 的多头），它们的差值很小但包含关键定价信息——灾难性抵消会**完全抹去**这个价差信号。实际中使用**Kahan 求和**算法或 **Decimal** 高精度库。

## 二、根求解：牛顿法（Newton's Method）

### 2.1 算法

对于方程 $f(x) = 0$，牛顿迭代公式：

$$
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
$$

二次收敛——每次迭代有效位数大致翻倍（当初始值足够好时）。

### 2.2 手算实例：用牛顿法求 $\sqrt{2}$

等价于求解 $f(x) = x^2 - 2 = 0$，$f'(x) = 2x$。

迭代公式：

$$
x_{n+1} = x_n - \frac{x_n^2 - 2}{2x_n} = \frac{x_n + 2/x_n}{2}
$$

从 $x_0 = 1.5$ 开始：

| 迭代 $n$ | $x_n$ | $f(x_n) = x_n^2 - 2$ | $f'(x_n) = 2x_n$ | 步长 $-f/f'$ | $x_{n+1}$ |
|----------|-------|----------------------|-------------------|--------------|-----------|
| 0 | 1.500000 | $2.25 - 2 = 0.2500$ | $3.0$ | $-0.2500/3.0 = -0.08333$ | 1.416667 |
| 1 | 1.416667 | $2.006944 - 2 = 0.006944$ | $2.8333$ | $-0.006944/2.8333 = -0.00245$ | 1.414216 |
| 2 | 1.414216 | $2.000006 - 2 = 0.000006$ | $2.8284$ | $-0.000006/2.8284 \approx -0.000002$ | 1.414214 |
| 3 | 1.414214 | $\approx 0$ | — | $0$ | 1.414214 |

**收敛速度**：从 1.5 到 1.416667（误差 $2 \times 10^{-3}$），再到 1.414214（误差 $< 10^{-6}$）——仅 3 次迭代就达到 6 位有效数字。

> **Quant Link**：牛顿法用于计算**隐含波动率**（IV）——求解 $C_{\text{BS}}(\sigma) - C_{\text{market}} = 0$，其中 $C_{\text{BS}}$ 是 Black-Scholes 的期权理论价格。$\sigma$ 的导数（vega）有解析公式，使得牛顿法高效精确。

## 三、数值积分

### 3.1 梯形法则（Trapezoidal Rule）

将 $[a, b]$ 均分为 $n$ 个小区间，每个小区间用梯形近似：

$$
\int_a^b f(x) dx \approx \frac{h}{2} \left[ f(a) + 2 \sum_{i=1}^{n-1} f(a + ih) + f(b) \right],\quad h = \frac{b-a}{n}
$$

误差：$\mathcal{O}(h^2)$。

### 3.2 手算实例：$\int_0^1 x^2 dx$ 用 4 个梯形

$n = 4$，步长 $h = (1 - 0)/4 = 0.25$。

| 节点 $i$ | $x_i$ | $f(x_i) = x_i^2$ | 权重 |
|----------|-------|-------------------|------|
| 0 | 0.00 | $0.00^2 = 0.0000$ | 1（端点） |
| 1 | 0.25 | $0.25^2 = 0.0625$ | 2 |
| 2 | 0.50 | $0.50^2 = 0.2500$ | 2 |
| 3 | 0.75 | $0.75^2 = 0.5625$ | 2 |
| 4 | 1.00 | $1.00^2 = 1.0000$ | 1（端点） |

$$
\begin{aligned}
\text{加权和} &= 1 \times 0 + 2 \times 0.0625 + 2 \times 0.2500 + 2 \times 0.5625 + 1 \times 1.0000 \\
&= 0 + 0.1250 + 0.5000 + 1.1250 + 1.0000 \\
&= 2.7500
\end{aligned}
$$

$$
\int_0^1 x^2 dx \approx \frac{0.25}{2} \times 2.7500 = 0.34375
$$

**理论值**：$\int_0^1 x^2 dx = \left[\frac{x^3}{3}\right]_0^1 = \frac{1}{3} \approx 0.33333$

**误差**：$0.34375 - 0.33333 = 0.01042$（约 3.13%）

> **Quant Link**：如果使用 **Simpson 法则**（权重模式 1-4-2-4-1），精度会提高到 $\mathcal{O}(h^4)$。在**蒙特卡洛定价**中，数值积分思想延伸为随机采样——高维积分（如亚式期权的路径积分）只能用 MC 方法。

## 四、Quant Link：数值 Greeks

### 4.1 有限差分法求 Delta

Delta $\Delta = \partial V / \partial S$，用**中心差分**数值近似：

$$
\Delta \approx \frac{V(S + \epsilon) - V(S - \epsilon)}{2\epsilon}
$$

其中 $\epsilon$ 的选择至关重要：
- $\epsilon$ 太大 → 截断误差（泰勒展开高阶项）
- $\epsilon$ 太小 → 灾难性抵消（浮点误差）

**最佳折衷**：$\epsilon \approx \sqrt{\varepsilon_m} \cdot S \approx 10^{-8} \times S$

### 4.2 有限差分 Delta 手算

假设 $S = 100$，$K = 100$，$r = 0.05$，$\sigma = 0.2$，$T = 1$ 年的看涨期权。

取 $\epsilon = 0.01$（$S$ 的 $0.01\%$）：

| 参数 | $S - \epsilon = 99.99$ | $S + \epsilon = 100.01$ |
|------|------------------------|-------------------------|
| BS 价格 $V$ | $10.4450$（示例值） | $10.4550$（示例值） |
| 差值 | — | $10.4550 - 10.4450 = 0.0100$ |
| 中心差分 | $\Delta \approx \frac{0.0100}{2 \times 0.01} = 0.5000$ |

解析 Delta = $N(d_1) \approx 0.5000$（平价附近）。中心差分 $\epsilon = 0.01$ 给出了 4 位精确的一致结果。

### 4.3 Monte Carlo 收敛速度

Monte Carlo 模拟的误差以 $\mathcal{O}(1/\sqrt{N})$ 收敛——要将精度提高 10 倍，模拟次数需要增加 **100 倍**。

| 模拟次数 $N$ | 标准误 $\sigma/\sqrt{N}$（$\sigma=0.2$ 为例） |
|-------------|----------------------------------------------|
| $10^2$ | $0.0200$ |
| $10^4$ | $0.0020$ |
| $10^6$ | $0.0002$ |
| $10^8$ | $0.00002$ |

> **Quant Link**：在**定价奇异期权**（如障碍期权、回望期权）时，Monte Carlo 与有限差分 Greeks 结合使用——MC 给出价格，路径上的有限差分估计 Greeks，称为**Pathwise Derivative Estimation**，比重抽法（bumping）更稳定。

## 五、Python 示例：有限差分 Delta

```python
import numpy as np
from scipy.stats import norm

def bs_call_price(S, K, r, sigma, T):
    """Black-Scholes 看涨期权价格"""
    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    return S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)

def finite_diff_delta(S, K, r, sigma, T, eps=0.01):
    """用中心差分计算数值 Delta"""
    V_up = bs_call_price(S + eps, K, r, sigma, T)
    V_down = bs_call_price(S - eps, K, r, sigma, T)
    return (V_up - V_down) / (2 * eps)

# 参数
S, K, r, sigma, T = 100.0, 100.0, 0.05, 0.2, 1.0

# 解析 Delta
d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
delta_analytic = norm.cdf(d1)
print(f"解析 Delta = {delta_analytic:.6f}")

# 数值 Delta（不同 epsilon）
for eps in [1.0, 0.1, 0.01, 0.001, 1e-6, 1e-10]:
    num_delta = finite_diff_delta(S, K, r, sigma, T, eps)
    err = abs(num_delta - delta_analytic)
    print(f"eps = {eps:12.0e}: 数值 Delta = {num_delta:.8f}, "
          f"误差 = {err:.2e}")

# --- 输出示例 ---
# 解析 Delta = 0.531261
# eps = 1e+00: 数值 Delta = 0.53126267, 误差 = 1.66e-06
# eps = 1e-01: 数值 Delta = 0.53126126, 误差 = 2.08e-09
# eps = 1e-02: 数值 Delta = 0.53126126, 误差 = 1.51e-11
# eps = 1e-03: 数值 Delta = 0.53126126, 误差 = 2.56e-12
# eps = 1e-06: 数值 Delta = 0.53126126, 误差 = 2.22e-10  ← 开始增大（灾难性抵消）
# eps = 1e-10: 数值 Delta = 0.53125898, 误差 = 2.28e-06  ← 严重恶化

# --- 牛顿法求 sqrt(2) ---
def sqrt_newton(x0=1.5, iters=4):
    x = x0
    print(f"n=0: x = {x:.10f}")
    for i in range(iters):
        x = (x + 2.0 / x) / 2.0
        print(f"n={i+1}: x = {x:.10f}, 误差 = {abs(x - np.sqrt(2)):.2e}")
    return x

sqrt_newton()
# --- 输出 ---
# n=0: x = 1.5000000000
# n=1: x = 1.4166666667, 误差 = 2.45e-03
# n=2: x = 1.4142156863, 误差 = 2.12e-06
# n=3: x = 1.4142135624, 误差 = 1.59e-12
```

**关键观察**：$\epsilon = 0.001$ 到 $0.01$ 是最优范围。$\epsilon$ 过小（$10^{-10}$）时，灾难性抵消使误差**重新增大**。

## 小结

| 主题 | 核心概念 | 量化金融应用 |
|------|---------|-------------|
| 浮点精度 | 机器 epsilon、灾难性抵消 | Greeks 计算中的 epsilon 选择 |
| 根求解（牛顿法） | $x_{n+1} = x_n - f/f'$ | 隐含波动率求解、IRR 计算 |
| 数值积分（梯形） | $\mathcal{O}(h^2)$ 精度 | 合约平均价格积分、利率曲线拟合 |
| Monte Carlo | $\mathcal{O}(1/\sqrt{N})$ 收敛 | 奇异期权定价、风险价值 (VaR) 估计 |
