# 09 数值计算

> 数值计算是连接数学理论和实际代码的桥梁。量化金融中绝大多数模型没有解析解——从期权定价到风险度量，全靠数值方法。理解浮点误差、收敛速度和数值稳定性，是写出可靠交易系统的前提。

---

## 章节内容

### [9.1 浮点精度](9.1-floating-point)
机器 epsilon 与灾难性抵消的定义，$(10^{16} + 1) - 10^{16}$ 的手算实例，以及大规模优化中的浮点问题。

### [9.2 根求解](9.2-root-finding)
二分法与牛顿法，$\sqrt{2}$ 的 3 次迭代手算表格，以及 IRR 和隐含波动率的计算。

### [9.3 数值积分](9.3-numerical-integration)
梯形法则与 Simpson 法则，$\int_0^1 x^2 dx$ 的 4 梯形手算实例，以及期权定价中的数值积分应用。

### [9.4 数值 Greeks](9.4-numerical-greeks)
有限差分法求 Delta/Gamma 的定义与 Python 实现，以及实际对冲中的数值方法。
