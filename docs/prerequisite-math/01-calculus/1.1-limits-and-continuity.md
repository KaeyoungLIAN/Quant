# 1.1 极限与连续

> 微积分的基石。极限定义了"趋近"的精确含义，连续则描述函数"没有断裂"的性质。

---

## 一、数列的极限

### 1.1.1 直观理解

考虑数列 $a_n = \frac{1}{n}$：

$$
a_1 = 1,\; a_2 = 0.5,\; a_3 = 0.333,\; a_4 = 0.25,\; \dots,\; a_{100} = 0.01,\; a_{1000} = 0.001
$$

当 $n$ 越来越大时，$a_n$ 越来越接近 $0$。我们说：**数列 $a_n$ 的极限是 $0$**，记作：

$$
\lim_{n \to \infty} \frac{1}{n} = 0
$$

### 1.1.2 精确定义（$\varepsilon$-$N$ 语言）

> 对任意 $\varepsilon > 0$，存在正整数 $N$，使得当 $n > N$ 时，有 $|a_n - A| < \varepsilon$。

用大白话说：**不论你给一个多小的误差范围 $\varepsilon$，我总能找到一个 $N$，之后的所有项都在这个误差范围内。**

**例**：证明 $\lim_{n\to\infty} \frac{1}{n} = 0$

- 给定 $\varepsilon > 0$，我们需要 $\frac{1}{n} < \varepsilon$，即 $n > \frac{1}{\varepsilon}$
- 取 $N = \lceil \frac{1}{\varepsilon} \rceil$，则当 $n > N$ 时，$|\frac{1}{n} - 0| = \frac{1}{n} < \varepsilon$
- 证毕 ✅

### 1.1.3 极限的性质

若 $\lim a_n = A$，$\lim b_n = B$，则：

| 性质 | 公式 |
|------|------|
| **线性** | $\lim (c_1 a_n + c_2 b_n) = c_1 A + c_2 B$ |
| **乘积** | $\lim (a_n b_n) = AB$ |
| **商** | $\lim \frac{a_n}{b_n} = \frac{A}{B}$（$B \neq 0$） |
| **夹逼** | $a_n \le c_n \le b_n$ 且 $\lim a_n = \lim b_n = L$ ⇒ $\lim c_n = L$ |

**夹逼定理示例**：证明 $\lim_{n\to\infty} \frac{\sin n}{n} = 0$

因为 $-1 \le \sin n \le 1$，所以 $-\frac{1}{n} \le \frac{\sin n}{n} \le \frac{1}{n}$

两边极限都是 $0$，由夹逼定理，中间极限也是 $0$。

## 二、函数的极限

### 2.1 定义

$$
\lim_{x \to a} f(x) = L
$$

含义：当 $x$ 无限接近 $a$（但不等于 $a$）时，$f(x)$ 无限接近 $L$。

### 2.2 左极限与右极限

| 概念 | 符号 | 含义 |
|------|------|------|
| **左极限** | $\lim_{x \to a^-} f(x)$ | 从左边趋近 $a$ |
| **右极限** | $\lim_{x \to a^+} f(x)$ | 从右边趋近 $a$ |

**存在条件**：$\lim_{x \to a} f(x)$ 存在 $\iff$ 左右极限都存在且相等。

**例**：$f(x) = \begin{cases} x^2, & x < 1 \\ 3, & x \ge 1 \end{cases}$

- $\lim_{x \to 1^-} f(x) = 1^2 = 1$
- $\lim_{x \to 1^+} f(x) = 3$
- 左右极限不相等 ⇒ $\lim_{x \to 1} f(x)$ **不存在**

### 2.3 无穷远极限

$$
\lim_{x \to \infty} f(x) = L
$$

含义：当 $x$ 越来越大时，$f(x)$ 趋近 $L$。

**例**：$\lim_{x \to \infty} \frac{1}{x} = 0$，$\lim_{x \to \infty} e^{-x} = 0$，$\lim_{x \to \infty} \arctan x = \frac{\pi}{2}$

### 2.4 无穷极限

$$
\lim_{x \to a} f(x) = \infty
$$

含义：当 $x$ 趋近 $a$ 时，$f(x)$ 无限增大。

**例**：$\lim_{x \to 0} \frac{1}{x^2} = \infty$（注意 $\frac{1}{x}$ 的左右极限方向不同）

## 三、极限的计算

### 3.1 直接代入

若 $f$ 在 $a$ 处连续（见下一节），直接代入 $x = a$。

$$ \lim_{x \to 2} (3x^2 + 1) = 3 \cdot 4 + 1 = 13 $$

### 3.2 因式分解（处理 $\frac{0}{0}$ 型）

$$ \lim_{x \to 2} \frac{x^2 - 4}{x - 2} = \lim_{x \to 2} \frac{(x-2)(x+2)}{x-2} = \lim_{x \to 2} (x+2) = 4 $$

### 3.3 有理化（处理根式）

$$ \lim_{x \to 0} \frac{\sqrt{1+x} - 1}{x}
= \lim_{x \to 0} \frac{(\sqrt{1+x} - 1)(\sqrt{1+x} + 1)}{x(\sqrt{1+x} + 1)}
= \lim_{x \to 0} \frac{x}{x(\sqrt{1+x} + 1)}
= \frac{1}{2} $$

### 3.4 重要极限

| 公式 | 说明 |
|------|------|
| $\displaystyle \lim_{x \to 0} \frac{\sin x}{x} = 1$ | 三角函数极限的基石 |
| $\displaystyle \lim_{x \to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2}$ | 由上式推导 |
| $\displaystyle \lim_{x \to \infty} (1 + \frac{1}{x})^x = e$ | 自然常数 $e$ 的定义 |
| $\displaystyle \lim_{x \to 0} \frac{e^x - 1}{x} = 1$ | 指数函数极限 |

### 3.5 洛必达法则

> 若 $\lim \frac{f(x)}{g(x)}$ 为 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型，则：

$$
\lim \frac{f(x)}{g(x)} = \lim \frac{f'(x)}{g'(x)}
$$

**例**：$\displaystyle \lim_{x \to 0} \frac{\cos x - 1}{x^2} = \lim_{x \to 0} \frac{-\sin x}{2x} = \lim_{x \to 0} \frac{-\cos x}{2} = -\frac{1}{2}$

> ⚠️ 使用前必须确认是 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型，且导数极限存在。

## 四、函数的连续性

### 4.1 定义

$f$ 在 $x = a$ 处连续 ⟺ 同时满足三个条件：

1. $f(a)$ 有定义
2. $\displaystyle \lim_{x \to a} f(x)$ 存在
3. $\displaystyle \lim_{x \to a} f(x) = f(a)$

**直观理解**：你可以一笔画过这个点，不用抬起笔。

### 4.2 间断点类型

| 类型 | 例子 | 示意图 |
|------|------|--------|
| **可去间断** | $\frac{x^2-1}{x-1}$ 在 $x=1$ | 有个洞，补上就连续 |
| **跳跃间断** | $\lfloor x \rfloor$ 在整数点 | 阶跃，左右极限不相等 |
| **无穷间断** | $\frac{1}{x}$ 在 $x=0$ | 趋近无穷 |
| **震荡间断** | $\sin\frac{1}{x}$ 在 $x=0$ | 剧烈震荡，极限不存在 |

### 4.3 连续函数的性质

- 连续函数的和、差、积、商（分母不为零）仍连续
- 连续函数的复合函数连续
- **介值定理**：若 $f$ 在 $[a,b]$ 上连续，则 $f$ 能取到 $f(a)$ 和 $f(b)$ 之间的所有值
- **最值定理**：若 $f$ 在 $[a,b]$ 上连续，则 $f$ 在该区间上能取到最大值和最小值

### 4.4 量化应用：二分法找零

介值定理的经典应用——找方程的根。

假设我们想找到某资产的隐含波动率 $\sigma$ 使得期权价格 $C(\sigma) = 16.5$：

```python
def find_implied_vol(target_price, low=0.01, high=2.0, tol=1e-6):
    """
    二分法找隐含波动率
    应用介值定理：C(low) < target < C(high) 时，中间必有解
    """
    while high - low > tol:
        mid = (low + high) / 2
        price = black_scholes_price(mid)  # 假设这个函数存在
        
        if price < target_price:
            low = mid
        else:
            high = mid
    
    return (low + high) / 2
```

## 五、Python 验证

### 极限计算符号验证

```python
import sympy as sp

n = sp.Symbol('n', integer=True, positive=True)
limit_1n = sp.limit(1/n, n, sp.oo)
print(f"lim 1/n = {limit_1n}")  # 0

x = sp.Symbol('x')
limit_sinc = sp.limit(sp.sin(x)/x, x, 0)
print(f"lim sin(x)/x = {limit_sinc}")  # 1
```

### 连续性可视化

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 1000)
y = np.sin(x) / x

plt.figure(figsize=(10, 5))
plt.plot(x, y, label=r'$\frac{\sin x}{x}$', linewidth=2)
plt.axhline(0, color='gray', linestyle='--', alpha=0.5)
plt.axvline(0, color='gray', linestyle='--', alpha=0.5)
plt.scatter([0], [1], color='red', s=100, zorder=5, label='可去间断点 (x=0)')
plt.title(r'$\lim_{x \to 0} \frac{\sin x}{x} = 1$', fontsize=14)
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

## 六、量化 & ML 应用

| 应用场景 | 联系 |
|----------|------|
| **牛顿法求根**（期权 IV 计算） | 利用函数极限和连续性，迭代逼近解 |
| **梯度下降收敛性** | 学习率过大可能导致损失函数震荡不收敛（类似震荡间断） |
| **激活函数** | Sigmoid / Tanh 在定义域内处处连续，这是梯度可计算的前提 |
| **资产价格极限行为** | 连续复利 $Ae^{rt}$ 就是 $\lim_{n\to\infty} A(1+\frac{r}{n})^{nt}$ |

## 小结

| 概念 | 要点 |
|------|------|
| 极限 | 趋近但不等于，$\varepsilon$-$N$ 或 $\varepsilon$-$\delta$ 定义 |
