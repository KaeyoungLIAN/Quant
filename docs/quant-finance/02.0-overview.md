# 02.0 组合管理与风险

> 本章回答量化金融最核心的问题之一：**如何科学地构建和管理一个投资组合？** 从单个资产的收益风险度量，到多资产组合的最优化，再到绩效评估和因子归因。

---

## 本章学习路线

```
组合收益与风险（基础度量工具）
    ↓
有效前沿与最优组合（Markowitz 框架）
    ↓
夏普比率与绩效度量（评价组合好坏）
    ↓
因子模型（归因收益来源）
    ↓
下一步：衍生品与定价
```

---

## 你将从本章学到什么

| 节号 | 内容 | 核心模型/工具 |
|:----:|------|:-------------:|
| 2.0 | 组合收益与风险 | 简单/对数收益、组合方差、相关系数 |
| 2.1 | 有效前沿与最优组合 | Markowitz 均值-方差优化 |
| 2.2 | 夏普比率与绩效度量 | Sharpe、Sortino、Calmar、信息比 |
| 2.3 | 因子模型 | CAPM、Beta、Fama-French 三因子 |
