# 04.0 策略与回测

> 本章从**策略构思**到**回测验证**到**绩效评估**，走完一个完整策略的生命周期。你会发现：好策略不是"赚钱就行"，而是"知道为什么赚钱、什么时候不赚钱、以及如何检验它是否可靠"。

---

## 本章学习路线

```
趋势跟踪策略（动量策略的经典实现）
    ↓
均值回归策略（反趋势策略）
    ↓
回测框架与方法论（如何避免虚假信号）
    ↓
绩效评估指标（CAGR、MDD、Sharpe 手算）
    ↓
下一步：市场微观结构
```

> **关键思维**：回测的终极目的是**证伪**，而不是**证明**。如果回测看起来太好，多半是哪里出了问题——数据窥探（Data Snooping）、前视偏差（Look-Ahead Bias）、幸存者偏差（Survivorship Bias）都是常见的坑。
