## 一、高等数学（微积分）

### 1.1 极限与连续

- 数列和函数的极限定义
- 无穷小与无穷大
- 连续函数的性质
- **量化应用**：资产价格的极限行为、收敛性分析

### 1.2 导数与微分

- 导数的定义与几何意义
- 求导法则（链式法则、乘积法则）
- 泰勒展开
- 偏导数与全微分
- **ML 应用**：梯度下降的核心工具
- **量化应用**：期权希腊值（Delta、Gamma）的计算

### 1.3 积分学

- 不定积分与定积分
- 积分技巧（换元、分部积分）
- 多重积分
- **量化应用**：计算期望值、风险度量
- **应用**：概率密度函数的归一化、期望计算

### 1.4 多元微积分

- 梯度向量
- 雅可比矩阵
- 海森矩阵
- **ML 应用**：多元优化、牛顿法、梯度下降的收敛性分析

## 二、线性代数

### 2.1 向量与矩阵

- 向量空间、线性无关
- 矩阵运算与性质（转置、求逆、迹）
- 范数（L1、L2、无穷范数）
- **ML 应用**：数据表示为向量，权重为矩阵
- **量化应用**：投资组合的权重向量与收益矩阵

### 2.2 线性方程组

- 高斯消元法
- 矩阵的秩与逆
- 最小二乘法
- **应用**：线性回归的闭式解

### 2.3 正定矩阵与二次型

- 正定、半正定矩阵的定义
- 二次型：$x^T A x$
- 检验方法：特征值全正、Cholesky 分解
- **量化应用**：协方差矩阵是半正定的；投资组合方差 $w^T \Sigma w$ 是二次型
- **ML 应用**：核函数、马氏距离

### 2.4 特征值分解与奇异值分解

- 特征值与特征向量
- 矩阵对角化
- 奇异值分解（SVD）
- **应用**：PCA、矩阵压缩、推荐系统

### 2.5 主成分分析（PCA）

- 通过协方差矩阵的特征分解找到主成分方向
- 方差解释比例
- **量化应用**：降维、风险因子模型、PCA 提取市场因子

### 2.6 矩阵微积分

- 矩阵对标量的导数
- 标量对向量的导数
- **ML 应用**：梯度推导、神经网络反向传播的矩阵形式

## 三、概率论

### 3.1 概率空间与条件概率

- 样本空间、事件、概率测度
- 条件概率与独立性
- 全概率公式、贝叶斯公式
- 全期望公式：$E[E[X|Y]] = E[X]$
- **量化应用**：事件驱动的交易策略概率建模

### 3.2 随机变量与分布

- 离散与连续随机变量
- 常用分布：正态、均匀、指数、伯努利、二项、泊松、t 分布、卡方分布、F 分布
- 分布函数与密度函数
- 联合分布、边缘分布、条件分布
- **量化应用**：资产收益率通常假设为正态分布（但实际有厚尾）

### 3.3 期望、方差、协方差矩阵、条件期望

- 期望与方差的性质（线性性）
- 协方差与相关系数
- **协方差矩阵**: 多变量间的协方差汇总
- 偏度与峰度（正态偏度=0, 峰度=3）
- 矩母函数与特征函数
- 条件期望：$E[X|Y]$，最小二乘预测中的核心概念
- **量化应用**：资产的风险（方差）与收益（期望）的权衡；
  投资组合方差 $w^T \Sigma w$

### 3.4 常用不等式

- **切比雪夫不等式**：$P(|X-\mu| \ge k\sigma) \le 1/k^2$
- 马尔可夫不等式
- 霍夫丁不等式
- **量化应用**：风险管理中的尾部概率上界估计

### 3.5 大数定律与中心极限定理

- 大数定律：样本均值收敛到期望
- 中心极限定理：和的分布趋近正态
- **量化应用**：蒙特卡洛模拟的理论基础、回测结果的统计显著性检验

## 四、数理统计

### 4.1 参数估计

- 点估计：矩估计、极大似然估计（MLE）
- 区间估计：置信区间
- 估计量的评价：无偏性、有效性、一致性
- **量化应用**：估计资产的预期收益率和波动率

### 4.2 假设检验

- 原假设与备择假设
- p 值与显著性水平
- t 检验、卡方检验、F 检验
- 第一类与第二类错误、检验功效（Power）
- **量化应用**：回测中检验策略是否显著优于基准

### 4.3 方差分析（ANOVA）

- 基本思想：将总变异分解为组间变异 + 组内变异
- F 统计量：$F = \frac{MS_{between}}{MS_{within}}$
- 单因素 ANOVA、多因素 ANOVA（简介）
- **量化应用**：比较多个策略/因子组的收益均值是否显著不同

### 4.4 回归分析

#### 4.4.1 线性回归与 OLS

- 线性回归模型：$y = X\beta + \varepsilon$
- 普通最小二乘法（OLS）：$\hat{\beta} = (X^TX)^{-1}X^Ty$
- 高斯-马尔可夫定理：OLS 是最佳线性无偏估计（BLUE）

#### 4.4.2 拟合优度与模型评估

- **R²（决定系数）**：
  - 定义：$R^2 = 1 - \frac{SS_{res}}{SS_{tot}} = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$
  - 含义：模型解释的变异占总变异的比例，值域 [0, 1]
  - 问题：增加自变量 R² 必然提高，需要用**调整 R²** 惩罚模型复杂度
  - **调整 R²**：$\bar{R}^2 = 1 - \frac{SS_{res}/(n-k-1)}{SS_{tot}/(n-1)}$（k 为自变量个数）
  - **量化应用**：因子模型（如 Fama-French）中 R² 衡量因子对收益的解释力度

- **F 检验**：检验所有回归系数是否同时为零（模型整体显著性）
- **残差分析**：残差 vs 拟合值图、Q-Q 图检验正态性

#### 4.4.3 回归诊断

- **异方差（Heteroscedasticity）**：
  - 定义：残差的方差不恒定（如金融数据中波动聚集）
  - 检测：Breusch-Pagan 检验、残差 vs 拟合值散点图
  - 后果：系数估计仍无偏但标准误有偏，t 检验不可靠
  - 修复：稳健标准误（White/Huber-White）、加权最小二乘（WLS）
  - **量化应用**：股票收益率的波动经常呈现异方差

- **自相关 / 序列相关（Autocorrelation / Serial Correlation）**：
  - 定义：$Cov(\varepsilon_i, \varepsilon_j) \neq 0$ 对 $i \neq j$，残差之间存在相关
  - **自相关函数（ACF）**：$\rho_k = \frac{Cov(y_t, y_{t-k})}{Var(y_t)}$
  - **偏自相关函数（PACF）**：排除中间滞后项影响后的纯滞后相关性
  - 检测：Durbin-Watson 检验（值≈2 无自相关，<1 正相关，>3 负相关）、
    ACF/PACF 图、Ljung-Box Q 检验
  - 后果：标准误被低估（t 统计量虚高），模型过拟合
  - 修复：Newey-West 稳健标准误、加入滞后项（AR 结构）、差分处理
  - **量化应用**：资产收益率序列的自相关检验（市场有效性检验），因子模型残差的自相关

- **多重共线性（Multicollinearity）**：
  - 定义：自变量之间存在强线性关系
  - 检测：方差膨胀因子（VIF），VIF > 5~10 视为严重
  - 后果：系数估计不稳定、标准误膨胀、符号可能反转
  - 修复：删除相关变量、PCA 降维、岭回归（L2 正则化）
  - **量化应用**：多因子模型中因子之间可能存在相关性

#### 4.4.4 正则化回归

- **岭回归（Ridge）**：L2 惩罚，压缩系数但不置零
- **Lasso**：L1 惩罚，可将不重要的系数压缩为零（特征选择）
- **弹性网（Elastic Net）**：L1 + L2 混合
- **量化应用**：因子筛选、高维数据建模（因子 zoo 问题）

### 4.5 时间序列分析

#### 4.5.1 基本概念

- **平稳性**：强平稳 vs 弱平稳（均值/方差/自协方差不随时间变化）
- **白噪声**：均值为零、方差恒定、无自相关的序列（模型残差理想状态）
- 自相关函数（ACF）与偏自相关函数（PACF）

#### 4.5.2 单位根检验

- **单位根**：时间序列的关键特征，$y_t = y_{t-1} + \varepsilon_t$ 有单位根 → 非平稳
- **ADF 检验**（Augmented Dickey-Fuller）：
  - $H_0$：序列有单位根（非平稳）
  - 若 p 值 < 0.05 → 拒绝 H₀，认为序列平稳
  - **量化应用**：检验资产价格的非平稳性（价格本身非平稳，收益率通常平稳）
- KPSS 检验（平稳性作为原假设，与 ADF 互补）

#### 4.5.3 经典模型

- AR(p)：自回归模型
- MA(q)：移动平均模型
- ARMA(p,q): 自回归移动平均
- ARIMA(p,d,q)：差分整合移动平均（d 为差分阶数）
- GARCH：波动率建模（条件异方差）
- **量化应用**：价格预测、波动率预测

#### 4.5.4 协整与配对交易

- **协整（Cointegration）**：两个或多个非平稳序列的线性组合是平稳的
- **Engle-Granger 两步法**：回归 → 对残差做 ADF 检验
- **量化应用**：配对交易策略的核心数学基础

## 五、最优化理论

### 5.1 无约束优化

- 梯度下降法
- 随机梯度下降（SGD）
- Adam、RMSProp（自适应学习率）
- **ML 应用**：模型训练的核心
- **量化应用**：参数调优

### 5.2 约束优化

- 等式约束：拉格朗日乘子法
- 不等式约束：KKT 条件
- **量化应用**：投资组合优化（Markowitz）
  - 公式：$\min_w w^T\Sigma w$ 满足 $w^T\mathbf{1}=1$，$w^T\mu = \mu_p$

### 5.3 凸优化

- 凸集与凸函数
- 凸优化问题的性质（局部最优 = 全局最优）
- **应用**：很多 ML 和投资组合问题可以表述为凸优化
