# Quant Wiki

> 量化金融 & AI 技术知识库

系统化整理量化交易、人工智能、技术栈的笔记与资源。教材级前置数学知识，配 Python 示例和手算推导。

🌐 **在线地址**：https://quant-9am.pages.dev/

---

## 内容结构

| 板块 | 内容 |
|------|------|
| 📈 量化金融 | 策略模型、回测框架、风险管理 |
| 🔢 前置数学知识 | 10 大板块、28 篇：高等数学、线性代数、概率论、数理统计、最优化、随机过程、信息论、傅里叶分析、数值计算、图论 |
| 🤖 AI 技术与工具 | LLM、AI Agent、RAG |
| 🛠️ 技术栈 | Python 生态、Django、React、数据库、部署运维 |
| 📚 资源收藏 | 书籍、论文、博客、数据集索引 |

## 技术栈

- [VitePress](https://vitepress.dev/) — 静态网站生成
- [KaTeX](https://katex.org/) — 数学公式渲染
- [Cloudflare Pages](https://pages.cloudflare.com/) — 部署托管

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build
```
